# Setup Instructions

Follow the steps below to run the Waste Classification project.

## Step 1: Clone the Repository
Download the project from GitHub.

## Step 2: Install Required Libraries
Install the required Python libraries using the requirements file.

pip install -r requirements.txt

## Step 3: Open the Project
Open the project in Python or Google Colab.

## Step 4: Run the Model
Run the Python script to load the trained model and classify waste images.

## Step 5: Upload an Image
Provide a waste image as input and the system will predict the waste category.
